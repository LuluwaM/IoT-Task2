# IoT-Task2

Description of task :
Serial communication between two Arduino boards,so that when you press the Button on the first Arduino, the LED on the second Arduino will turn on.

Hardware :

- 2 Arduino Board
- 1 LED
- 1 Button
- 2 Resistor
- Hook-up wires
